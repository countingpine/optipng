OptiPNG: Advanced PNG Optimizer

Current version: OptiPNG 0.4.5 (released 31 May 2004)
Download the source code: optipng-0.4.5.tar.gz, optipng-0.4.5.zip
Download the Windows executable: optipng-0.4.5-exe.zip

OptiPNG is a PNG optimizer that recompresses the image files to a smaller size, without losing any information.
The idea has been inspired from pngcrush (http://pmt.sourceforge.net/pngcrush), and is explained in detail in the PNG-Tech article "A guide to PNG optimization". The implementation is carried forward in OptiPNG, which offers a faster execution per trial, and a wider search space.

NOTE:
The OptiPNG source distribution contains modified versions of zlib-1.2.1 and libpng-1.0.15. The original libraries are available at their home sites:

  • zlib at http://www.zlib.org/
    (A notable change: the constant TOO_FAR is set to 32768, for a slight compression improvement; see the PNG-Tech article "TOO_FAR in zlib is not too far...".)
  • libpng at http://www.libpng.org/pub/png/libpng.html
    (Although OptiPNG is designed to work with any recent version of libpng (post- 1.0.6), we distribute a special version that contains some corrections and improvements.)
IMPORTANT:
Comments and complaints about these library distributions should NOT be addressed to the authors or to the maintainers of the original libraries.
You may use the original libraries instead of those supplied in the source package; however, if you use the supplied ones, it is strongly recommended that you link them in statically.
